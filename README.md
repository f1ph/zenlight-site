# ZENLIGHT™ Landing Page

Página estática pronta para GitHub + Cloudflare Pages.

## Arquivos

- `index.html` — estrutura da página
- `style.css` — visual, responsividade e paleta
- `script.js` — animações simples e scroll suave
- `assets/` — imagens e vídeo do produto

## Como subir no GitHub

1. Crie um repositório chamado `zenlight-site`.
2. Envie todos estes arquivos para a raiz do repositório.
3. Na Cloudflare Pages, conecte o repositório.
4. Configuração de build:
   - Framework preset: `None`
   - Build command: deixar vazio
   - Output directory: `/` ou deixar vazio, dependendo da interface
5. Publique.
